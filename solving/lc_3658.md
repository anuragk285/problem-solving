# Find GCD of odd Even sums

```python
class Solution(object):
    def gcdOfOddEvenSums(self, n):
        """
        :type n: int
        :rtype: int
        """
        # mathematically ans = n
        even_sum = n*(n+1)
        odd_sum = n**2

        def gcd(a, b):
            if b == 0 : return a
            return gcd(b, a%b)

        return gcd(even_sum, odd_sum)
```
