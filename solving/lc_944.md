# Delete columns to make sorted

```java
class Solution {
    public int minDeletionSize(String[] strs) {
        int deletions = 0, i =0;
        while(i<strs[0].length()){
            for(int j=1; j<strs.length; j++){
                if((strs[j-1].charAt(i)-'a') > (strs[j].charAt(i)-'a')){
                    deletions++;
                    break;
                }
            }
            i++;
        }
        return deletions;
    }
}
```
