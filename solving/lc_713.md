# Subarray product lessthan K

```python
class Solution(object):
    def numSubarrayProductLessThanK(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: int
        """
        if k <= 1:
            return 0
        product = 1
        ans = 0
        j = 0
        for i in range(len(nums)):
            product *= nums[i]
            while product >= k and j<len(nums):
                product //= nums[j]
                j += 1
            ans += i - j + 1

        return ans
```
