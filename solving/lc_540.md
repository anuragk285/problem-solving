# Single element in a sorted array

```python
class Solution(object):
    def singleNonDuplicate(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        i = 0
        while i+1 < len(nums):
            if nums[i] != nums[i+1]: return nums[i]
            i = i+2
        return nums[-1]
```
