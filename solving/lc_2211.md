# Count Collisions on a Road

```java
class Solution {
    public int countCollisions(String directions) {
        int last = -1, ans = 0;
        for(char c : directions.toCharArray()){
            if(c == 'L'){
                if(last >= 0){
                    ans += last + 1;
                    last = 0;
                }
            }else if(c == 'S'){
                if(last > 0){
                    ans += last;
                }
                last = 0;
            }else {
                if(last < 0) last = 1;
                else last++;
            }
        }
        return ans;
    }
}
```
