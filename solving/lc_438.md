# Find all anagrams in a String

```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        if(s.length() < p.length()) return new ArrayList<Integer>();
        int[] mainFreq = new int[26];
        for(char c : p.toCharArray()) mainFreq[c-'a']++;
        int[] freq = new int[26];
        int i = 0;
        while(i < p.length()){
            char c = s.charAt(i++);
            freq[c-'a']++;
        }
        int j = i;
        i = 0;
        ArrayList<Integer> ans = new ArrayList<>();
        if(Arrays.equals(mainFreq, freq)) ans.add(0);
        while(j < s.length()){
            char cj = s.charAt(j), ci = s.charAt(i);
            freq[cj-'a']++;
            freq[ci-'a']--;
            if(Arrays.equals(mainFreq, freq)) ans.add(i+1);
            i++;
            j++; 
        }
        return ans;
    }
}
```
