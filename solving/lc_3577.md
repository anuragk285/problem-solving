# Count the number of Computer Unlocking Permutation

```java
class Solution {
    public int countPermutations(int[] complexity) {
        long ans = 1;
        for(int i=1; i<complexity.length; i++){
            if(complexity[0] >= complexity[i]) return 0;
        }
        long mod = 1_000_000_007;
        for(int i=1; i<complexity.length; i++){
            ans = (ans*i)%mod;
        }
        return (int)ans;
    }
}
```
