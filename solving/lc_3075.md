# Maximum Happiness of selected children

```java
class Solution {
    public long maximumHappinessSum(int[] happiness, int k) {
        PriorityQueue<Integer> hap = new PriorityQueue<>((a, b) -> b - a);
        for(int h : happiness) hap.add(h);
        long sum = 0;
        for(int i=0; i<k; i++){
            int h = hap.poll();
            sum += Math.max(h-i, 0);
        }
        return sum;
    }
}
```
