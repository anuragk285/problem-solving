# Adjacent Increasing Subarrays Detection II

```python
class Solution(object):
    def maxIncreasingSubarrays(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        curRun, prevRun = 1, 0
        i = 1
        k = 0
        for i in range(1, len(nums)):
            if nums[i] > nums[i-1]:
                curRun += 1
            else:
                prevRun = curRun
                curRun = 1
            k = max(k, curRun//2)
            k = max(k, min(curRun, prevRun))
        return k
```
