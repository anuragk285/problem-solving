# Positions of Large Group

```java
class Solution {
    public List<List<Integer>> largeGroupPositions(String s) {
        List<List<Integer>> ans = new ArrayList<>();
        int i=0;
        while(i<s.length()){
            int start = i, end = i;
            char c = s.charAt(i);
            while(end < s.length() && c == s.charAt(end)) end++;
            if(end - start >= 3){
                List<Integer> l = new ArrayList<>();
                l.add(start);
                l.add(end-1);
                ans.add(l);
            }
            i = end;
        }
        return ans;
    }
}
```
