# License Key Formatting

```java
class Solution {
    public String licenseKeyFormatting(String s, int k) {
        String ans = "";
        int l = 0;
        for(int i=s.length()-1; i>=0; i--){
            char c = s.charAt(i);
            if(c == '-') continue;
            if(l == k){
                ans = "-" + ans;
                l = 0;
            }
            ans = Character.toUpperCase(c) + ans;
            l++;
        }
        return ans;
    }
}
```
