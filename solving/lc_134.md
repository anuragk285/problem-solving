# Gas Station

```python
class Solution(object):
    def canCompleteCircuit(self, gas, cost):
        """
        :type gas: List[int]
        :type cost: List[int]
        :rtype: int
        """
        if sum(cost) > sum(gas):
            return -1
        
        start, tank = 0, 0
        for i in range(len(gas)):
            tank = tank - cost[i] + gas[i]
            if tank < 0:
                tank = 0
                start = i+1
        return start
```
