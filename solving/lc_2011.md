# Find Value of variable after performing operations

```python
class Solution(object):
    def finalValueAfterOperations(self, operations):
        """
        :type operations: List[str]
        :rtype: int
        """
        ans = 0
        for oprtn in operations:
            if oprtn in ['++X', 'X++']:
                ans += 1
            elif oprtn in ['--X', 'X--']:
                ans -= 1
        return ans
```
