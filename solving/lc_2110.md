# Number of smooth Descent of a stock

```java
class Solution {
    public long getDescentPeriods(int[] prices) {
        if(prices.length == 1) return 1;
        long ans = 0;
        int i=0, j=1;
        while(j<prices.length){
            while(j<prices.length && prices[j] == prices[j-1]-1) j++;
            int n = j - i;
            ans += (long)n*(n+1)/2;
            i = j;
            if(j != prices.length-1) j++;
            else return ans+1;
        }       
        return ans;
    }
}
```
