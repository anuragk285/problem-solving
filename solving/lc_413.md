# Arithematic slices

```java
class Solution {
    public int numberOfArithmeticSlices(int[] nums) {
        if(nums.length < 3) return 0;
        int diff = -1, count = 0, ans = 0;
        for(int i=2; i<nums.length; i++){
            int a = nums[i-2], b = nums[i-1], c = nums[i];
            if(b-a == c-b){
                count = 0;
                diff = c - b;
                while(i<nums.length && nums[i] - nums[i-1] == diff){
                    count++;
                    i++;
                }
                ans += (count * (count + 1)) / 2;
            }
        }
        return ans;
    }
}
```
