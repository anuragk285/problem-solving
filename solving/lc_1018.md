# Binary Prefix divisible by 5

```java
class Solution {
    public List<Boolean> prefixesDivBy5(int[] nums) {
        List<Boolean> ans = new ArrayList<>();
        int n =0;
        for(int i=0; i<nums.length; i++){
            n = (n*2 + nums[i])%5;
            ans.add(n==0);
        }
        return ans;
    }
}
```
