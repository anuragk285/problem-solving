# Number of Steps to Reduce a Number in Binary Representation to One

```java
class Solution {
    public int numSteps(String s) {
        int count = 0, carry = 0;
        for(int i=s.length()-1; i>0; i--){
            int sum = ((int)(s.charAt(i)) + carry)%2;
            if(sum == 0) count++;
            else{
                carry = 1;
                count += 2;
            }
            System.out.println(sum);
        }
        return count + carry;
    }
}
```
