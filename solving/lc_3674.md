# Minimum operations to equalize array

```python
class Solution(object):
    def minOperations(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        all_equal = True
        for i in range(len(nums)): 
            if i>0 and nums[i] != nums[i-1]: all_equal = False
        if all_equal : return 0
        return 1
```
