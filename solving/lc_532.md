# K-diff Pairs in an Array

```java
class Solution {
    public int findPairs(int[] nums, int k) {
        HashMap<Integer, Integer> freq = new HashMap<>();
        for(int n : nums) freq.put(n, freq.getOrDefault(n, 0) + 1);
        int ans = 0;
        for(int n : freq.keySet()){
            if(k == 0){
                if(freq.get(n) >= 2) ans++;
            }else{
                if(freq.containsKey(n + k)) ans++;
            }
        }
        return ans;
    }
}
```
