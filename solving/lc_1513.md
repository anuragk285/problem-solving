# Number of substrings with only 1's

```java
class Solution {
    public int numSub(String s) {
        long mod = 1000000007, ans = 0;
        int i =0, j =0;
        while(j<s.length()){
            while(j<s.length() && s.charAt(j) == '0') j++;
            i = j;
            while(j<s.length() && s.charAt(j) == '1') j++;
            j--;
            int n = j - i + 1;
            ans = (ans + ((long)n * (n+1)/2))%mod; 
            j++;
        }
        return (int)ans;
    }
}
```
