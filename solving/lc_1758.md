# Minimum Changes To Make Alternating Binary String

```java
class Solution {
    public int minOperations(String s) {
        String starting0 = generate(s.length(), '0');
        String starting1 = generate(s.length(), '1');
        int s1=0, s2=0;
        for(int i=0; i<s.length(); i++){
            if(starting0.charAt(i) != s.charAt(i)) s1++;
            if(starting1.charAt(i) != s.charAt(i)) s2++;
        } 
        return (int)Math.min(s1, s2);
    }
    private String generate(int n, char c){
        String ans = "" + c;
        for(int i=1; i<n; i++){
            if(ans.charAt(ans.length()-1) == '0') ans += '1';
            else ans += '0';
        }
        return ans;
    }
}
```
