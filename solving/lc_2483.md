# Minimum Penalty for a Shop

```java
class Solution {
    public int bestClosingTime(String customers) {
        int n = customers.length();
        int[] leftEmpty = new int[n+1];
        int[] rightRush = new int[n+1];
        int empty = 0;
        for(int i=0; i<=n; i++){
            leftEmpty[i] = empty;
            if(i<n && customers.charAt(i) == 'N') empty++;
        }
        int rush = 0;
        rightRush[n] = 0;
        for(int i=n-1; i>=0; i--){
            if(customers.charAt(i) == 'Y') rush++;
            rightRush[i] = rush;
        }
        int ans = -1, minPenalty = Integer.MAX_VALUE, profit = 0;
        for(int i=0; i<=n; i++){
            int penalty = leftEmpty[i] + rightRush[i];
            if(penalty < minPenalty){
                ans = i;
                minPenalty = penalty;
            }
        }
        return ans;
    }
}
```
