# Minimum time visiting all points

```java
class Solution {
    public int minTimeToVisitAllPoints(int[][] points) {
        int time = 0;
        for(int i=1; i<points.length; i++){
            int currX = points[i-1][0], currY = points[i-1][1];
            int nextX = points[i][0], nextY = points[i][1];
            int timeX = 0, timeY = 0;

            // travel diag for X-cor
            int diag_travel = Math.abs(currX - nextX);
            int straight_travel = Math.abs(Math.abs(currY - nextY)-diag_travel);
            timeX = diag_travel + straight_travel;

            //travel diag for Y-cor
            diag_travel = Math.abs(currY - nextY);
            straight_travel = Math.abs(Math.abs(currX - nextX) - diag_travel);
            timeY = diag_travel + straight_travel;
        
            time += timeX < timeY ? timeX : timeY;
        }
        return time;
    }
}
```
