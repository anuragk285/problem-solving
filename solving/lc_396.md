# Rotate function

```java
class Solution {
    public int maxRotateFunction(int[] nums) {
        int n = nums.length, sum =0, ans = Integer.MIN_VALUE;
        int[] f = new int[n];
        for(int v : nums) sum += v;
        for(int i=0; i<n; i++){
            if(i == 0){
                f[0] = 0;
                for(int j=0; j<n; j++) f[0] += j*nums[j];
            }else{
                f[i] = f[i-1] + sum - (n*nums[n-i]);
            }
            ans = Math.max(ans, f[i]);
        }
        return ans;
    }
}
```
