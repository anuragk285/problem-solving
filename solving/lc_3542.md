# Minimum operations to convert All elements to Zero

```java
class Solution {
    public int minOperations(int[] nums) {
        Stack<Integer> stack = new Stack<>();
        int count =0;
        for(int n : nums){
            while(!stack.empty() && stack.peek() > n) stack.pop();
            if(n == 0) continue;
            if(stack.empty() || stack.peek() < n){
                count += 1;
                stack.push(n);
            }
        }
        return count;
    }
}
```
