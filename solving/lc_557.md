# Reverse words in a String III

```java
class Solution {
    public String reverseWords(String s) {
        int i = 0;
        String ans = "";
        while(i<s.length()){
            String word = "";
            while(i<s.length() && s.charAt(i) != ' '){
                word = s.charAt(i) + word;
                i++;
            }
            ans += word;
            while(i<s.length() && s.charAt(i) == ' '){
                ans += ' ';
                i++;
            }
        }
        return ans;
    }
}
```
