# Smallest Integer Divisible by k

```java
class Solution {
    public int smallestRepunitDivByK(int k) {
        int rem = 0, count =0;
        if(k%2 == 0 || k%5 == 0) return -1;
        HashSet<Integer> remainders = new HashSet<>();
        do{
            rem = rem*10 + 1;
            count++;
            rem = rem%k;
            if(remainders.contains(rem)) return -1;
            remainders.add(rem);
        }while(rem != 0);
        return rem == 0 ? count : -1;
    }
}
```
