# Optimal partition of a string

```python

class Solution(object):
    def partitionString(self, s):
        """
        :type s: str
        :rtype: int
        """
        count = 0
        sub_str = ""
        
        for i in range(len(s)):
            if s[i] in sub_str :
                count += 1
                sub_str = s[i]
            else :
                sub_str += s[i]
            print(sub_str, i, count)
        return count+1                
```
