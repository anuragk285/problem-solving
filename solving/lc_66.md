# Plus One

```java
class Solution {
    public int[] plusOne(int[] digits) {
        int carry = 1, i = digits.length-1;
        while(carry > 0 && i >= 0){
            int val = digits[i] + carry;
            digits[i] = val % 10;
            carry = val / 10;
            i--;
        }
        if(carry == 1){
            int[] ans = new int[digits.length+1];
            ans[0] = 1;
            for(int j=1; j<digits.length+1; j++) ans[j] = digits[j-1];
            return ans;
        }
        return digits;
    }
}
```
