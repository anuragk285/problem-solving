# No of zero filled sub arrays

```python
class Solution(object):
    def zeroFilledSubarray(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        ans = 0
        zeros = 0
        for i in nums:
            if i == 0:
                zeros += 1
                ans += zeros
            else : zeros = 0
        return ans
```
