# Reconstruct Original Digits from English

```java
class Solution {
    public String originalDigits(String s) {
        int[] digits = new int[10];
        digits[0] = count(s, 'z');
        digits[2] = count(s, 'w');
        digits[4] = count(s, 'u');
        digits[6] = count(s, 'x');
        digits[8] = count(s, 'g');
        digits[1] = count(s, 'o') - digits[0] - digits[2] - digits[4];
        digits[3] = count(s, 'h') - digits[8];
        digits[5] = count(s, 'f') - digits[4];
        digits[7] = count(s, 's') - digits[6];
        digits[9] = count(s, 'i') - digits[6] - digits[8] - digits[5];
        String ans = "";
        for(int i=0; i<=9; i++){
            ans += Integer.toString(i).repeat(digits[i]);
        }
        return ans;
    }
    private int count(String s, char ch){
        int ans =0;
        for(char c : s.toCharArray()) if(c == ch) ans++;
        return ans;
    }
}
```
